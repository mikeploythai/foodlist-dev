# FoodList (WIP)

FoodList is an open source food inventory management solution. This was created for a full-stack web app tutorial (coming soon).

**Stack**

- Next.js 14
- TypeScript
- Supabase
- CSS Modules
- Radix UI Primitives (plus cmdk)
- Tabler Icons
- Zod
- Posthog (maybe)
- Vercel
